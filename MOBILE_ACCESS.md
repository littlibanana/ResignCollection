# 手機訪問指南

## 如何從手機訪問網站

### 步驟 1：確保電腦和手機在同一網絡

- 確保您的手機和電腦連接到同一個 Wi-Fi 網絡
- 如果使用手機熱點，請確保電腦連接到該熱點

### 步驟 2：啟動開發服務器

在項目目錄中執行：

```bash
npm run dev
```

### 步驟 3：查找電腦的 IP 地址

#### Windows 系統：

1. 按 `Win + R`，輸入 `cmd`，按 Enter
2. 在命令提示符中輸入：`ipconfig`
3. 找到「IPv4 地址」，例如：`192.168.1.100`

#### 或者使用 PowerShell：

```powershell
ipconfig | findstr IPv4
```

### 步驟 4：在手機瀏覽器中訪問

在手機瀏覽器中輸入：

```
http://您的IP地址:5173
```

例如：`http://192.168.1.100:5173`

**注意**：請將 `您的IP地址` 替換為步驟 3 中找到的實際 IP 地址。

### 步驟 5：如果無法訪問

1. **檢查防火牆**：
   - Windows 可能會阻止連接
   - 在防火牆設置中允許端口 5173

2. **檢查端口**：
   - 如果 5173 端口被占用，Vite 會自動使用下一個可用端口
   - 查看終端輸出中的實際端口號

3. **確保服務器正在運行**：
   - 終端應該顯示類似：`Local: http://localhost:5173/`
   - 和 `Network: http://192.168.x.x:5173/`

✅ **響應式設計**：自動適配手機屏幕  
✅ **觸摸友好**：按鈕和輸入框大小適合觸摸操作

✅ **快速加載**：優化的樣式和佈局  
✅ **PWA 支持**：可以添加到主屏幕（通過瀏覽器選項）

## 添加到主屏幕（iOS）

1. 在 Safari 中打開網站
2. 點擊底部的「分享」按鈕
3. 選擇「加入主畫面」
4. 點擊「加入」

## 添加到主屏幕（Android）

1. 在 Chrome 中打開網站
2. 點擊右上角的三點菜單
3. 選擇「加入主畫面」或「安裝應用」
4. 確認添加

## 故障排除

### 問題：手機無法連接到服務器

**解決方案**：
- 確保手機和電腦在同一 Wi-Fi 網絡
- 檢查 Windows 防火牆是否阻止了連接
- 嘗試關閉 VPN（如果正在使用）

### 問題：頁面顯示不正常

**解決方案**：
- 清除瀏覽器緩存
- 嘗試使用不同的瀏覽器
- 確保使用最新版本的瀏覽器

### 問題：輸入框自動縮放（iOS）

**解決方案**：
- 已經在代碼中設置了 `font-size: 16px` 來防止此問題
- 如果仍有問題，請檢查瀏覽器設置

## 生產環境部署

如果您想讓網站永久在線，可以：

1. **使用 Vercel**（推薦）：
   ```bash
   npm install -g vercel
   vercel
   ```

2. **使用 Netlify**：
   - 將項目推送到 GitHub
   - 在 Netlify 中連接倉庫
   - 自動部署

3. **使用自己的服務器**：
   ```bash
   npm run build
   # 將 dist 目錄部署到您的服務器
   ```

部署後，您就可以從任何地方訪問網站了！
